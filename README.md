## Usage

1. Clone this repository locally
2. Open the terminal and `cd` to the cloned repository
3. Type the following command:
```
sudo cp -r src/* /usr/share/supertuxkart/data/skins
```
4. Open SuperTuxKart and go to Options > Interface
5. Go to the Skin selector and select your flavor.
